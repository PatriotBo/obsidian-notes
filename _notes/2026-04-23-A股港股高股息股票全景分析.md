## 一、什么是"真高股息"？

并非所有高股息率都值得投资。**真高股息**需要满足：

| 条件 | 说明 | 反例 |
|------|------|------|
| ✅ 连续3年以上稳定分红 | 不是一次性分红推高的 | 达安基因20.3%（一次性） |
| ✅ 派息率 40%-80% | 分红有余力，不是掏空家底 | 思维列控263%（不可持续） |
| ✅ 业务现金流充沛 | 分红有"真金白银"支撑 | 部分地产股借钱分红 |
| ✅ 市值 ≥ 100亿 | 有流动性保障 | 创业板微盘股（流动性陷阱） |
| ✅ PE > 0 且合理 | 不是亏损导致的"高股息" | 周期底部亏损公司 |

## 七、⚠️ 高股息陷阱警示

以下类型的"高股息"需要警惕：

| 陷阱类型 | 特征 | 案例 |
|---------|------|------|
| **一次性分红** | 突然超高派息率，往往是掏老底 | 达安基因(20.3%)、思维列控(12.4%) |
| **股价暴跌推高** | 股息率高是因为股价跌了，不是分红多了 | 部分地产股、创业板小票 |
| **周期顶部高分红** | 景气顶部利润最高，分红最多，之后就下去了 | 部分航运股、周期性煤炭股 |
| **借钱分红** | 自由现金流为负还在分红 | 高负债地产公司 |
| **微盘股高息** | 流动性差，卖不出去白搭 | 创业板海王英特龙(95%!) |

### 验证方法

```
✅ 连续3年以上分红记录
✅ 派息率 < 80%（非一次性）
✅ 自由现金流 > 分红金额
✅ 市值 ≥ 100亿（流动性保障）
✅ 主营业务利润稳定或增长
```

## 💡 个人思考

- 高股息策略在当前低利率环境下价值凸显，5%+股息率远超银行存款和理财产品
- 港股免税标的（中国移动、汇丰、REITs）是收息性价比之王
- 高股息不等于低成长——中国神华、招商银行2026年涨幅均超10%，股息+资本增值双收
