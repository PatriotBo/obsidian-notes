# 产品审查：高考方向推荐 AI 助手

> 来源：个人产品规划
> 审查日期：2026-04-14 | 模式：创意打磨 - 商业线

## 📌 核心要点

- **产品代号**：志选 (ZhiXuan)，取"智慧选择"之意
- **一句话定位**：一个链接、一份问卷、一份 AI 报告——帮信息弱势的高考家庭搞清楚"适合什么方向"
- **核心差异化**：不做出分后的志愿填报（夸克/百度已做），只做出分前的方向探索（市场空白）
- **目标用户**：农村及三四线城市高考家庭，通过熟人分享链接触达
- **技术方案**：Next.js + Vercel + DeepSeek API，1-2 周可上线 MVP
- **综合评分**：7.5/10，值得花 2 周做出来验证

## 📝 详细笔记

### 一、产品概述

**核心定位**：不是填志愿工具（出分后用），而是**方向探索工具**（出分前就能用）。通过结构化问卷摸底考生画像，AI 生成个性化的专业/方向推荐报告，降低信息差。

**使用场景**：亲戚/朋友收到求助 → 分享链接 → 考生/家长填问卷 → AI 自动生成推荐报告 → 拿报告做决策或结合夸克/百度等工具精准填报。

**核心用户流程**：
```
熟人分享链接 → 打开(零注册零下载) → 填问卷(10-15min) → AI生成报告(30-60s)
                                                             ↓
                                              总结版(1页精华) + 详细报告(完整分析)
                                                             ↓
                                              行动建议：去夸克/百度精准填报
```

### 二、市场调研

#### 市场规模
| 指标 | 数据 |
|------|------|
| 2025 年高考报名人数 | **1335 万**（历史新高） |
| 志愿填报付费市场规模 | **10.2 亿元**（2024），预计 2025 年超 10.9 亿 |
| 考生付费意愿 | **88.1%** 愿意为志愿填报服务付费 |
| 新高考改革省份 | **29 个省**已落地，填报复杂度大增 |

#### 竞品全景

**第一梯队：互联网巨头（免费）**
| 产品 | 背景 | 用户量 | 核心定位 | 与我们的关系 |
|------|------|--------|---------|-------------|
| 夸克高考 | 阿里 | 1.2 亿+ | 出分后智能填志愿 | **互补** |
| 百度 AI 志愿助手 | 百度 | 单日千万 | 出分后查学校 | **互补** |
| 阳光志愿 | 教育部 | 未披露 | 官方权威数据 | **互补** |

**第二梯队：AI 创业产品**
| 产品 | 形态 | 收费 | 与我们的关系 |
|------|------|------|-------------|
| 高考纸鸢 | 微信小程序 | 免费 | **部分竞争** |

**第三梯队：人工咨询**
- 张雪峰/峰学蔚来：17999 元/人，2026 年已售罄
- 一般志愿规划师：3000-8000 元，2025 年业务量腰斩
- 亲戚朋友（我帮外甥的场景）：免费但信息有限、费时费力

#### 关键洞察：市场空白

```
出分前（方向探索）     →  ❌ 几乎空白（只有张雪峰人工做，1.8万起）
出分后（志愿填报）     →  ✅ 夸克/百度/纸鸢/靠谱（已成熟）

我们的产品 → 切的是"出分前"这个空白！
```

### 三、产品设计

#### 问卷设计（4 大模块 15 题）

**模块一：基本信息**（省份/选科/成绩/排名）
**模块二：家庭背景**（经济条件/学费承受/家长期望）
**模块三：个人特质**（性格/兴趣/排除项/身体条件/特殊情况）
**模块四：未来规划**（就业地域/稳定vs高薪/深造意愿）

#### AI 报告设计

**总结版**（给家长/亲戚快速看）：考生画像 + TOP 3 推荐方向 + 建议避开 + 下一步行动
**详细版**（完整分析）：
1. 考生画像全景（性格分析、兴趣图谱、能力雷达图）
2. 推荐方向详解（5-8 个方向，含学什么/做什么/薪资/风险/院校）
3. 避开方向及原因
4. 升学路径建议
5. 给家长的话
6. 下一步行动清单

### 四、技术方案

| 选择 | 理由 |
|------|------|
| **Next.js** | 正在学的技术栈，SSR 对 SEO 友好，Vercel 免费部署 |
| **Vercel** | 免费额度够 MVP 用，自动 HTTPS |
| **DeepSeek API** | 国产模型，便宜（百万 token 约 1-2 元），中文能力强 |
| **Supabase** | 免费 PostgreSQL，存问卷和报告数据 |
| **备选** | 微信小程序（第二阶段） |

### 五、方案对比

| 方案 | 工作量 | 风险 | 推荐 |
|------|--------|------|------|
| A: 纯网页 MVP（Next.js+Vercel） | 1-2 周 | 低 | ⭐ **推荐** |
| B: 微信小程序 | 2-3 周 | 中 | 第二阶段 |
| C: H5+微信公众号 | 1-2 周 | 低 | 备选 |

**推荐方案 A**：速度最快、技术栈匹配、成本几乎为零

### 六、风险矩阵

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| AI 报告质量不稳定 | 高 | 高 | 精心设计 Prompt + 报告模板 |
| 用户不信任AI建议 | 中 | 高 | 报告说明"仅供参考"，引导结合其他工具 |
| Vercel 国内访问不稳 | 中 | 中 | 备选 Cloudflare Pages |
| DeepSeek API 不稳定 | 低 | 高 | 备选 Claude/GPT API |
| 微信封链接 | 低 | 高 | 自定义域名+备案 |

### 七、执行路线图

**Phase 1 MVP（1-2 周）**：问卷+报告生成+部署，目标 10 个真实用户
**Phase 2 迭代（3-4 周）**：优化 Prompt、分享功能、打赏功能，目标 100 用户
**Phase 3 增长（2-3 月）**：小程序版本、内容营销、付费试水，目标 1000+ 用户

⏰ **时间窗口**：2026 高考 6 月 7 日，需在 5 月初上线赶上这波

### 八、导师观察

1. ✅ 找到了真实的"代理人"场景——"农村家长找有学历的亲戚咨询"是真实行为模式
2. ✅ "不做填志愿"的克制很对——避开巨头正面战场
3. ⚠️ 真正用户是"被求助的亲戚"，不是"家长"，产品体验应围绕此设计
4. ⏰ 时间窗口紧——需在 4 月底/5 月初上线

## 💡 个人思考

- 这是独立开发的第二个产品方向（第一个是"吵架帮手"），起源于帮外甥做高考志愿指南的真实经历
- 核心竞争力不在技术（LLM 谁都能调），而在 Prompt 工程和报告质量——这是需要持续迭代的
- 商业模式暂不急，先做出来跑用户量，高考季是最好的验证窗口
- 如果验证成功，这个产品有年度复用性（每年高考季都有需求）

### 报告模版要点提炼（供产品 Prompt 设计参考）

**报告生成 Prompt 需要覆盖的关键能力：**

1. **数据检索能力**：根据省份+选科+分数，检索匹配的院校和专业录取数据
2. **成绩诊断能力**：分析各科强弱势，给出"适合/避开"方向建议
3. **专业解读能力**：用通俗语言解释"学什么→做什么→赚多少"
4. **院校推荐能力**：按冲/稳/保梯度，给出具体院校+分数线+王牌专业
5. **结构化输出**：总结版（1页给家长）+ 详细版（完整报告给考生）
6. **风险提示**：分数线波动、录取变动等不确定性的免责说明

**报告质量标准：**
- 语言通俗易懂，避免专业术语（目标用户可能没受过高等教育）
- 每个推荐都有具体理由（为什么适合这个考生）
- 薪资、就业率等关键数据要有来源或说明
- 提供明确的行动建议（不是只给方向，还要告诉怎么做）
